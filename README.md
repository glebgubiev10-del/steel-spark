# СтальИскра — Интернет-магазин строительных инструментов

## Технологии
- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS 4
- shadcn/ui
- Zustand (состояние + корзина)

## Локальный запуск
```bash
bun install
bun run dev
```

## Деплой на Vercel
Проект готов к деплою. Просто подключите репозиторий к Vercel — никаких дополнительных настроек не требуется.

Корзина хранится в localStorage через Zustand persist, внешний сервер не нужен.
